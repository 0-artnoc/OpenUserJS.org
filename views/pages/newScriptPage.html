Script Metadata

You can read about most userscript metadata blocks on the greasespot wiki and at the greasemonkey sourceforge wiki.

OpenUserJS.org Specific

@author {{#authedUser}}{{authedUser.name}}{{/authedUser}}{{^authedUser}}username{{/authedUser}}

The author of the script. Required for Collaboration. Only the author may edit.

@collaborator username

Can be defined multiple times. Required for Collaboration.

@description This script even does the laundry!

Specially formatted on the script page.

@copyright Year, Author (Author Homepage)

[Wiki Link] Specially formatted on the script page.

@icon url

A url or data url. Used in the script list page at 16px and on the script page at ~45px.

@license License Type; License Homepage

[Wiki Link] Specially formatted on the script page.

Collaboration

To allow other users to upload modified versions of your script to your account, add @author Zren to the metadata of your script, along with @collaborator username for every user you wish to give write access. Only the real author of the script can modify these metadata keys. Collaborators cannot use GitHub integration to update the script. Add them as collaborators to the GitHub repo instead.
